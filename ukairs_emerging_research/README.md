# UKAIRS Emerging Research Submission

This folder contains an Overleaf-friendly LaTeX scaffold for a UKAIRS Emerging Research / Provocation submission.

## Files

- `main.tex`: ACM `sigconf` two-column submission scaffold.
- `references.bib`: placeholder references to verify before submission.

## Notes

- Target length: 2 main pages. References may be on an additional page, based on the UKAIRS guidance.
- Use the ACM/Overleaf template with `\documentclass[sigconf]{acmart}`.
- Every figure should include `\Description{...}` for accessibility.
- Replace all `TODO` fields before submission.
- Verify author names, publication dates, and citation metadata before final export.

## Suggested Overleaf Sync

The cleanest option is to sync this folder as its own Overleaf project. If you use Git, either:

1. Keep this folder inside the current PhD/VLM repo while drafting, then upload or sync only this folder to Overleaf.
2. Create a separate submission repo later if you want Overleaf Git sync to track only paper files.

For the current stage, keeping it inside this repo is simpler because the submission depends on your existing research notes.
