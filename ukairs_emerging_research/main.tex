\begin{abstract}
Medical vision-language models (VLMs) are able to answer questions about clinical images, yet it remains unclear whether their generated explanations are faithful to the visual evidence used for decision-making. This concern becomes more important in agentic VLM systems, where memory, tool use, and iterative reasoning can create plausible but weakly grounded evidence chains. This research uses gastrointestinal (GI) endoscopy as a case study to ask how VQA models and potential agentic workflows should be audited before they are treated as evidence-grounded clinical decision support. Building on preliminary fine-tuning and explanation extraction for GI VQA, the proposed framework evaluates final answers, intermediate tool outputs, memory updates, uncertainty estimates, and multimodal explanations using calibration analysis, counterfactual evidence, grounding consistency, and clinician-facing tasks. The intended contribution is a reusable evaluation protocol and framework for characterising grounding and reasoning risks in safe, evidence-grounded agentic medical AI.
\end{abstract}

\section{Motivation}

Gastrointestinal (GI) endoscopy is a visually demanding and safety-critical clinical setting in which diagnostic decisions depend on identifying and interpreting subtle mucosal abnormalities under substantial variation in anatomy, viewpoint, illumination, image quality, bowel preparation, and procedural artefacts~\cite{beg2017quality}. This project aims to contribute to an evaluation framework for auditing evidence faithfulness in agentic medical VLM workflows, as well as a framework for characterising grounding and reasoning risks involving ungrounded tool use, misleading memory updates, and post-hoc rationales.


\section{Related Work and Gap}

GI endoscopy VQA has recently been formalised through datasets and shared tasks such as Kvasir-VQA, and Medico 2025, which provide image-question-answer pairs for evaluating medical reasoning over gastrointestinal imagery~\cite{gautam2024kvasirvqa,gautam2025kvasirvqa,gautam2025medico}. Early experimental results suggest that domain adaptation and parameter-efficient fine-tuning can improve performance on such tasks~\cite{peter2026parameterefficient}. In parallel, medical VLM benchmarks such as MedHEval and recent studies of medical visual grounding show that hallucinations and weak image grounding remain central risks for medical VLM systems~\cite{chang2025medheval,liu2026medical}.

Recent agentic medical-imaging systems, including EndoCogniAgent and other iterative VLM frameworks, extend single-pass reasoning with memory, tool use, retrieval, routing, reflection, and repair~\cite{tang2025endocogniagent,alim2026gaze,sayeedi2026route}. However, agency introduces new failure surfaces because an agent may store unsupported observations, retrieve them as evidence, select an inappropriate tool, misinterpret a tool output, or preserve a misleading intermediate hypothesis~\cite{dash2026untrusted,zhang2026memmorph}.

Emerging challenge-style evaluations are consistent with a broader agent-evaluation literature that emphasises sequential clinical decision-making, incomplete information, tool use, and realistic EHR-style environments~\cite{jiang2025medagentbench,schmidgall2024agentclinic}. This motivates hidden stress tests in which structured reasoning traces are judged against incomplete, longitudinal, and sometimes conflicting multimodal evidence, with unsupported or hallucinated claims penalised even when the final prediction is plausible. Existing evaluations still largely emphasise task accuracy, benchmark performance, or hallucination rates. Less is known about whether intermediate agent states, tool outputs, explanations, and memory updates remain faithful to the underlying visual evidence. This creates a safety-critical gap for clinical AI where an agent can produce coherent reasoning while accumulating ungrounded evidence.

\section{Research Questions}

This project is guided by three questions. First, how do standalone and agentic VLM systems for GI endoscopy balance answer correctness with clinically coherent multimodal explanations? Second, are the visual highlights, textual explanations, and structured reasoning traces produced by these systems faithful to the image evidence and clinically relevant question context? Third, do uncertainty estimates and confidence-calibration signals reflect answer correctness, explanation faithfulness, missing evidence, conflicting evidence, and corrupted intermediate tool outputs or memory updates?

\section{Proposed Framework}

The proposed workflow compares a standalone VLM baseline with an agentic VLM pipeline for GI endoscopy. The standalone baseline receives an image or video frame and a clinical question, then produces an answer, uncertainty estimate, and optional explanation. The agentic pipeline additionally supports planned evidence acquisition steps such as lesion localisation, segmentation, retrieval of similar cases, explanation generation, uncertainty-aware validation, and final answer selection.

The key object of analysis is not only the final answer, but the full evidence chain. Each intermediate step can be audited for whether it is visually grounded, semantically consistent with the question, appropriately uncertain, and causally relevant to the final decision. This is particularly important to understand if memory, reflection, and tool use can preserve or amplify an initially ungrounded observation. The planned audit will also test whether agents acknowledge missing evidence, resolve conflicts between tool outputs, express uncertainty when evidence is insufficient, and avoid importing unsupported findings into the reasoning trace.

\section{Evaluation Plan}

The evaluation will follow the answer-and-explanation structure suggested by emerging GI VQA challenges and datasets~\cite{gautam2025kvasirvqa,gautam2025medico}. First, final-answer performance will be measured using task and language metrics such as accuracy, precision, recall, F1 score, and agreement with reference answers across question types including yes/no, single-choice, multiple-choice, location-related, colour-related, and numerical-count questions. This establishes whether the standalone and agentic systems solve the base VQA task before explanation quality is considered.

Second, multimodal explanations will be assessed for clarity, coherence, medical relevance, consistency with visual evidence, and behavioural safety. Textual justifications will be checked for factual correctness, appropriate uncertainty, and adherence to clinical best practice. Visual explanations will be linked to clinically meaningful terms, such as \emph{polyp}, \emph{ulcer}, \emph{lesion}, and \emph{bleeding}, and compared with available bounding-box, segmentation, or region-level annotations. This tests whether an explanation is not only plausible, but grounded in the image regions relevant to the question.

To move beyond final outputs, the agentic setting will additionally require a structured reasoning trace for each case. Guided by agent-evaluation literature on sequential clinical reasoning and tool-assisted decision-making, each trace will record which evidence was used, which tool outputs were accepted or rejected, how uncertainty was represented, and how missing or conflicting evidence was handled~\cite{jiang2025medagentbench,schmidgall2024agentclinic}. Unsupported, contradictory, or hallucinated claims in the trace will be treated as reasoning-risk events even when the final answer is correct.

Uncertainty quantification will be evaluated as a process-level safety signal, following established approaches for estimating predictive uncertainty in deep learning~\cite{kendall2017uncertainties}. The study will examine whether confidence, predictive entropy, answer disagreement across repeated or ensemble-style runs, and selective abstention behaviour track answer correctness, explanation faithfulness, missing information, and conflicting evidence. Calibration will be assessed using reliability analysis and proper scoring rules, and uncertainty should increase when the available evidence is ambiguous, insufficient, or inconsistent with the reasoning trace.

Robustness to agent-level errors will be evaluated by corrupting intermediate evidence. Example interventions include shifting a segmentation mask, altering a region proposal, replacing a retrieved case with an irrelevant example, or injecting conflicting evidence into memory. The analysis will determine whether the agent detects inconsistencies, seeks corroborating evidence, acknowledges missing information, or incorporates corrupted evidence uncritically.

Finally, a clinician-facing human evaluation will compare answer-only, visual-explanation, and multimodal-explanation conditions. The study will measure whether explanations improve users' ability to identify incorrect model outputs, calibrate confidence appropriately, and avoid inappropriate reliance on plausible but unfaithful explanations.


\section{Current Status}

Preliminary work has explored GI polyp segmentation with U-Net and transformer-based models, together with visual explanation and uncertainty-estimation tools. This earlier pipeline motivates the shift from pixel-level segmentation evaluation toward uncertainty-aware, evidence-grounded VQA and agentic reasoning. Current work has begun with a PaliGemma-based GI VQA pipeline. A notebook fine-tunes \texttt{google/paligemma-3b-pt-224} on Kvasir-VQA-x1 using QLoRA/LoRA, 4-bit quantisation, a frozen vision encoder, and sampled validation for resource-constrained experimentation. Initial inference checks use a LoRA adapter for GI question answering, while an explanation wrapper extracts attention maps, Grad-CAM-style saliency, token-level confidence scores, and medical terminology from generated answers. These components are preliminary infrastructure for the proposed audit; they do not yet constitute a completed clinical validation. The next stage is to convert this baseline into an agentic evidence-validation workflow and evaluate whether each reasoning step remains grounded in clinically relevant visual evidence.
